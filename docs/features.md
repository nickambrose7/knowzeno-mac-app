# Onboarding
- When the user first downloads the app, there should be an onboarding flow where:
    1. We ask for their API key. 
    2. Let them choose a default keyboard shortcut for sending a note to the knowzeno backend
    3. Explain to them what this app does.
    
    
# App Interface
The main app interface will be very simple. It simply needs to display the text that the user indicate
