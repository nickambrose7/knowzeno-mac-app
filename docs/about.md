The purpose of this Mac app is very simple. I want to be able to highlight any text on my computer. It could be in the browser, a notes app, Obsidian, anywhere that I can select text. I want to be able to left click on that text or press a keyboard shortcut and which would then send that text to this app. For now, once we get access to that highlighted text in this app, just print it.


## Notes
- This app will ask to control the computer via the accessablity settings. This must be enabled by the user, and we should have a guide to show the user how to get everything working.
